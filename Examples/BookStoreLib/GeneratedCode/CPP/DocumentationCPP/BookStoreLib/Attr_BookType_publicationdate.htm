LtXmlLib21::CDateTime GetPublicationdate() const;
void SetPublicationdate(LtXmlLib21::CDateTime val);
  Property Description  
    Property Name publicationdate  
    Property Type LtXmlLib21::CDateTime  
    Parent Class BookStoreLib::CBookType  
    XML Type Attribute  
    Accessors Read/Write  
    Description An optional CDateTime  
    Remarks It is optional, initially it is not populated.
The property IsValidPublicationdate indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidPublicationdate = false.

 
    Also See IsValidPublicationdate - Indicates if the property has been populated.  
    Example 
BookStoreLib::CBookTypePtr spParent = ...;

// Because the property is Optional, it will NOT initially be populated
// Attempting to read from it WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidPublicationdate flag.
ASSERT(spParent->IsValidPublicationdate() == false);

// So it can be set as follows 
LtXmlLib21::CDateTime newItem;
newItem.SetDate(2005, 11, 29);
spParent->SetPublicationdate(newItem);

// As the property is populated IsValidPublicationdate will now return true
ASSERT(spParent->IsValidPublicationdate() == true);

// So it can be read  as follows 
printf(_T("Object Value (%s)\n"), spParent->GetPublicationdate()->ToString().c_str());
