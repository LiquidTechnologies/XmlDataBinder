java.lang.String getISBN
  Property Description  
    Property Name getISBN  
    Property Type java.lang.String  
    Parent Class BookStoreLib.BookType  
    XML Type Attribute  
    Accessors Read/Write  
    Description An optional java.lang.String  
    Remarks It is optional, initially it is not populated.
The property isValidISBN indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property set setValidISBN = false.

 
    Also See isValidISBN - Indicates if the property has been populated.  
    Example 
BookStoreLib.BookType parent = ...;

// Because the property is Optional, it will NOT initially be populated
// Attempting to read from it WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the isValidISBN flag.
assert parent.isValidISBN() == false : "Initially the property is not populated";

// So it can be set as follows 
parent.setISBN("Sample Value");

assert parent.isValidISBN() == true : "As the property is populated isValidISBN will now return true";

// So it can be read  as follows 
java.lang.String val = parent.getISBN();
